✅ Чтобы обновить репозиторий на GitHub через терминал в Visual Studio, следуйте этим шагам:

1. **Откройте терминал** в Visual Studio. Для этого нажмите `Ctrl + ~` или используйте меню `View` -> `Terminal`.
2. **Перейдите в каталог проекта**. Используйте команду `cd`, чтобы переместиться в нужную директорию с вашим проектом:
   cd путь/к/вашему/проекту
3. **Проверьте статус репозитория**. Узнайте, какие файлы были изменены или добавлены:
   git status
4. **Добавьте изменения**. Чтобы добавить все изменения в индекс (стадию), выполните: 
   git add .   
   или добавьте конкретный файл:
   git add "имя_файла"
5. **Коммит изменений**. Сделайте коммит с описанием изменений:
   git commit -m "Описание изменений"
6. **Получите последние изменения из удаленного репозитория** (если нужно синхронизировать локальный репозиторий с удаленным):
   git pull origin main
   (Замените `main` на `master` или другую ветку, если у вас используется другая ветка).
7. **Отправьте изменения на GitHub**. После того как коммит сделан, отправьте его на сервер:
   git push origin main

Дополнительные команды, которые могут быть полезны:
- **Просмотр истории коммитов**: git log
- **Создание новой ветки**: git checkout -b имя_новой_ветки
- **Переключение на другую ветку**: git checkout имя_выбранной_ветки (например, main)
- **Удаление ветки**: git branch -d имя_выбранной_ветки
- **Удаление локальной ветки**: git branch -D имя_выбранной_ветки
- **Удаление удалённой ветки**: git push origin --delete имя_выбранной_ветки
- **Восстановление удалённой ветки**: git push origin имя_выбранной_ветки
- **Установка логина и пароля**:
   git config --global user.email "you@example.com"
   git config --global user.name "Your Name"
